# BookShop API

This is a backend project for a book shop.

Users can:

- Sign up
- Sign in
- Create books
- Update books
- Delete books
- Buy books
- See their books
- Browse the public catalog

## Technologies

- Node.js
- Express
- TypeScript
- Prisma
- PostgreSQL
- JWT
- Jest

## Installation

Install the dependencies:

```bash
npm install
```

Create a `.env` file.

Run the database:

```bash
npx prisma migrate dev
```

Start the project:

```bash
npm run dev
```

## Run tests

```bash
npm test
```

## Build

```bash
npm run build
```
